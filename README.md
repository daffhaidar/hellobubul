# 🐾 Hellobubul

> Platform Terpercaya untuk Pet Parents Indonesia

[![Live Site](https://img.shields.io/badge/site-live-brightgreen)](https://hellobubul.vercel.app)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Next.js](https://img.shields.io/badge/Next.js-14-black?logo=next.js)](https://nextjs.org/)
[![Deployed on Vercel](https://img.shields.io/badge/Deployed%20on-Vercel-black?logo=vercel)](https://vercel.com/)

**Hellobubul** adalah platform yang membantu pet parents di Indonesia menemukan:
- 🏥 Vet terverifikasi dengan review transparan
- 💊 Obat dan vitamin original (garansi 100%)
- 🏠 Adopsi aman & terjamin
- 💬 Telekonsultasi dengan dokter hewan terpercaya
- 👥 Komunitas pecinta anabul yang supportive

🔗 **Live Site:** [hellobubul.vercel.app](https://hellobubul.vercel.app)

---

## 🎯 Vision

Membangun ekosistem pet care yang terpercaya di Indonesia, dimana setiap pet parent bisa:
- Mendapatkan akses ke vet berkualitas tanpa ribet
- Terhindar dari obat palsu yang merugikan
- Menemukan komunitas yang genuinely care tentang kesejahteraan anabul

**Didirikan oleh pemilik 8 kucing yang paham betul struggle pet parents Indonesia.** 🐱

## 🛠️ Tech Stack

### Frontend
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Font:** Plus Jakarta Sans + Inter
- **Icons:** Custom SVG

### Backend & Services
- **Hosting:** Vercel (auto-deploy on push)
- **Forms:** Formspree (waitlist submissions)
- **Database:** Supabase (coming soon)
- **Analytics:** Vercel Analytics
- **SEO:** Next.js Metadata API + JSON-LD

### Development
- **Language:** TypeScript
- **Package Manager:** npm
- **Version Control:** Git + GitHub
- **CI/CD:** Vercel Auto-Deploy
