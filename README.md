# 🐾 Hellobubul

Platform Terpercaya untuk Anabul Indonesia

> Temukan vet terbaik, obat original, dan komunitas pecinta anabul dalam satu platform.

## 🎯 Vision

Membantu pet owners di Indonesia mendapatkan:
- ✅ Vet terverifikasi (no more vet palsu!)
- ✅ Obat 100% original (garansi uang kembali)
- ✅ Adopsi aman & transparan (anti penipuan & penganiayaan)
- ✅ Komunitas yang supportive

## 🚀 Status

**MVP Phase** - Target launch: July 2026

- [x] Landing page + waitlist
- [ ] Vet directory
- [ ] Marketplace MVP
- [ ] Telemedicine
- [ ] Community features

## 🛠️ Tech Stack

- **Frontend:** Next.js 14 + Tailwind CSS
- **Hosting:** Vercel (free tier)
- **Database:** Supabase (free tier)
- **Forms:** Formspree (free tier)
- **Analytics:** Vercel Analytics

## 🏃 Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Deploy to Vercel
vercel --prod
```
